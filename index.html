Abstract

Document layout analysis is essential for downstream tasks such as information retrieval, extraction, OCR, and digitization. Existing datasets often lack fine-grained labels or multilingual diversity, limiting research on Hebrew manuscripts. We introduce HePU, a foundational dataset designed for semantic script segmentation in Hebrew paleography. This dataset provides high-quality annotations across diverse manuscript scripts and aims to support model development for layout understanding, classification, and digitization of historical Hebrew texts.
